2. Lei Geral de Proteção de Dados (LGPD)

Em conformidade com a Lei nº 13.709/2018 (LGPD), garantimos a proteção dos seus dados pessoais:

2.1 Coleta de Dados

Coletamos os seguintes tipos de dados:

  • Dados de identificação (nome, e-mail, telefone)
  • Dados de localização (para busca de objetos próximos)
  • Dados de transação (histórico de aluguéis)
  • Dados de navegação (cookies e analytics)

2.2 Uso dos Dados

Seus dados são utilizados para:

  • Fornecer e melhorar nossos serviços
  • Processar transações e reservas
  • Comunicar sobre atualizações e ofertas
  • Prevenir fraudes e garantir segurança

2.3 Seus Direitos

Acesso aos seus dados pessoais
Correção de dados incompletos ou desatualizados
Anonimização, bloqueio ou eliminação de dados
Portabilidade dos dados
Revogação do consentimento
